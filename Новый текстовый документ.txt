Перейдите в каталог с вашим проектом:

Используйте команду cd для перехода в каталог с вашим проектом. Например:
bash
Copy code
cd путь/к/вашему/проекту
Запустите веб-сервер:

После перехода в каталог с проектом, выполните следующую команду:
bash
Copy code
http-server
По умолчанию, веб-сервер будет запущен на порту 8080.
Откройте браузер:

Откройте ваш веб-браузер и перейдите по адресу http://localhost:8080 (или другой порт, который указан в выводе команды).
